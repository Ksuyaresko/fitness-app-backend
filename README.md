# fitness-app-backend

## Swagger

https://goit-be.onrender.com/api-docs/

## Description
This is a backend for fitness app.
Stack: node.js express.js, mongodb, mongoose, jsonwebtoken, cloudinary, nodemailer

To start the express server locally, run the following

`npm run start:dev`
